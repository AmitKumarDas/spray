cc.spray.http

HttpRequest

case class HttpRequest (method: HttpMethod, uri: String, headers: List[HttpHeader], content: Option[HttpContent], protocol: HttpProtocol) extends HttpMessage[HttpRequest] with Product with Serializable

Sprays immutable model of an HTTP request. The uri member contains the the undecoded URI of the request as it appears in the HTTP message, i.e. just the path, query and fragment string without scheme and authority (host and port).
